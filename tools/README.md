# tools/ — Asset Pipeline

Raw Kenney kits (zips + extracted folders) live in `tools/raw-assets/` — **git-ignored**, only
processed output enters the repo under `public/assets/`.

## Commands

```sh
npm run assets:build   # process all assets listed in tools/assets.config.mjs
```

## Files

| File | Role |
| --- | --- |
| `assets.config.mjs` | Which raw files get processed, with category/loadGroup/license metadata. **Add new assets here.** |
| `process-assets.mjs` | Pipeline: dedup → prune → weld → meshopt (`EXT_meshopt_compression`) → optional KTX2 (needs `toktx` on PATH) → `public/assets/` + manifest. Prints before/after sizes. |
| `raw-assets/` | Raw downloads (git-ignored). Re-download URLs are in `docs/2026-06-12-kenney-inventory.md`. |

## Runtime requirement

GLBs are meshopt-compressed: three.js `GLTFLoader` must get the decoder —
`loader.setMeshoptDecoder(MeshoptDecoder)` from the `meshoptimizer` package.

## Manifest schema (`public/assets/manifest.json`)

Generated by the pipeline — **do not edit by hand**; edit `assets.config.mjs` and rebuild.
An asset that is not in the manifest must not be loaded by the game.

```jsonc
{
  "version": 1,            // manifest schema version
  "generated": "YYYY-MM-DD",
  "ktx2": false,           // whether textures were KTX2-compressed in this build
  "assets": [
    {
      "id": "bed-single",                  // unique kebab-case id used by loader code
      "file": "assets/models/furniture-beds/bedSingle.glb", // URL path from site root
      "type": "model",                     // model | texture | audio (audio: OGG+MP3 pair, lesson L3)
      "category": "beds",                  // GDD 04-content category or "room-shell"
      "loadGroup": "furniture-beds",       // lazy-load unit; "core" = preloaded FTUE set
      "source": {                          // provenance (CLAUDE.md license rule)
        "origin": "kenney",                // kenney | generated
        "pack": "Furniture Kit",
        "packVersion": "1.0",
        "url": "https://kenney.nl/assets/furniture-kit"
      },
      "license": "CC0-1.0",                // SPDX id
      "bytes": 12345                       // processed size on disk
    }
  ]
}
```

### Lazy-load groups

`core` is the only group fetched before first interaction (FTUE room, budget: first
payload < 5 MB gzip). All `furniture-*` groups load on demand when their catalog
category first opens (lesson L8).
